# Internet Link Repository

## Prerequisites

- Maven 3.3+
- JDK 1.8+

### Setup

After cloning this repository, fire up backend REST API:

```
$ cd backend
```

```
$ mvn spring-boot:run
```

To start front-end client:

```
$ cd frontend
```

```
$  yarn
```

```
$  yarn start
```

Yarn should open the web app in your default browser at http://localhost:3000

Enjoy!

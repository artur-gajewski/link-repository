# Internet Link Repository

## Prerequisites

- Maven 3.3+
- JDK 1.8+
- Yarn / npm

## Technologies

- Java 8
- Javascript / ES6
- Spring Boot
- React
- Styled Components

### Setup

After cloning this repository, fire up Spring Boot backend REST API:

```
$ cd backend
```

```
$ mvn spring-boot:run
```

To start React client:

```
$ cd client
```

```
$  yarn | npm install
```

```
$  yarn start | npm start
```

The app is now available at http://localhost:3000

Enjoy!
